Introduction

In this documentation, we are going to present one of the most used programming languages nowadays which is Python. We will present just some basics of this language and try to cover all the most basic statements. If you were curious and you want to know more about this language, please click on this link that will take you to the official documentation of python. Let's know more about this language.

Python is an intrepreted high-level programming language which means that this language doesn't need a compilation phase to execute the program and its keywords and statements are sort of close to human language and have some sort of abstraction from the machine language. This language provide different tools that allow you to write programs for different purposes like web development, machine learning or data visualization and other fields. One of the features that makes this language worth learning is that it follows an object-oriented approach that makes it easier to model some objects in our real world and write applications. It provides also the dynamic typing and garbage collection features which make coding easier. In addition to object-oriented programming, this language supports other programming paradigms such as Fonctional Programming. What makes this language easier and worth to learn is that its standard library conntains a lot of tools that allow you to make useful programs.

Guido van Rossum began working on Python in the late 1980's, as a successor to the ABC programming language, and first released it in 1991 as Python 0.9.1. Python 2.0 was released in 2000 and introduced new features, such as list comprehensions and a garbage collection system using reference counting and was discontinued with version 2.7.18 in 2020. Python 3.0 was released in 2008 and was a major revision of the language that is not completely backward-compatible, with much Python 2 code not running unmodified on Python 3.

Interactive Mode VS Script Mode

In Python, there are actually two ways to run a code which are :

  • Interactive Mode
  • Script Mode


The Intercative Mode, that is called also REPL, provides a quick way of running blocks or a single line of Python code. This code is executed via the Python shell that come with python installation. This mode is really useful when you want to just run some basic commands or you are new to Python programming. Here are some of the advantages of the Interactive Mode :

  1. Helpful when your script is extremely short and you want immediate results.
  2. Faster as you only have to type a command and then press the enter key to get the results.
  3. Good for beginners who need to understand Python basics.

But still there are some disadvantages of this mode too:

  1. Editing the code in interactive mode is hard as you have to move back to the previous commands or else you have to rewrite the whole command again.
  2. It's very tedious to run long pieces of code.


On the other hand, the Script Mode is used for writing long pieces of Python code or for Python Scripts that span multiple files. In this mode, you write your code in a text file and save it with the .py extension which stands for Python. You can use any text editor for that like Sublime Text, Atom or even the Notepad. Variables

Variables are containers for storing data values. There are some rules that should be respected in order to name these variables :

  • A variable name should start with a letter or an underscore character.
  • A variable name cannot start with a number.
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9 and _).
  • Variable names are case-sensitive.

Here are some examples of valid variable names : 

    
                    
                    x=5
                    myvar = "jhon"
                    _MyVar = "jhon"
                    Age = 20

These are invalid variable names : 

    
                    
                    5x=5
                    my var = "jhon"
                    My-Var = "jhon"

One of the features offered by python is that we can assign values to multiple variables in just one line. You should just make sure that the number of values matches the number of values to avoid an error. Here is an example : 

                    
                    x, y, z = "Orange", "Banana", "Cherry"
You could also assign one value to multiple variables : 
                    
                    x = y = z = "Orange"


Built-in Data Types

Variables can store data of different types, and different types can do different things. Python has the following data types built-in by default :

  • Text Type: str.
  • Numeric Type: int, float, complex.
  • Sequence Type: list, tuple, range.
  • Mapping Type: dict.
  • Set Type: set, frozenset.
  • Boolean Type: bool.
  • Binary Type: bytes, bytearray, memoryview.
In order to get the data type of a specific variable, you can use the type() function : 
                    
                    x=5
